Como lembrar de todas as senhas, e ainda garantir que tenham um nível aceitável de segurança?

<!--more-->

{% highlight ruby %}
def show
  @widget = Widget(params[:id])
  respond_to do |format|
    format.html # show.html.erb
    format.json { render json: @widget }
  end
end
{% endhighlight %}
